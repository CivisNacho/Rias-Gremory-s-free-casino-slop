## Getting Started

### Prerequisites

- Node.js (v20 or higher recommended)
- npm

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Create a production build in the `dist/` directory:
```bash
npm run build
```

## Deployment

This project is configured for **GitHub Pages**. Pushing to the `main` branch will automatically trigger the GitHub Actions workflow to build and deploy the application.
